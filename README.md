# 我的GitHub Pages网站

这是我的个人GitHub Pages网站仓库。

## 项目结构

```
.
├── index.html      # 主页面
├── css/           # CSS文件夹
│   └── style.css  # 样式文件
└── README.md      # 项目说明文件
```

## 如何使用

1. 克隆此仓库到本地
2. 修改 `index.html` 中的内容以适应您的需求
3. 修改 `css/style.css` 以自定义样式
4. 提交更改并推送到GitHub
5. 在仓库设置中启用GitHub Pages

## 部署

要部署此网站，请确保：

1. 仓库名称格式为：`用户名.github.io`
2. 在仓库设置中启用GitHub Pages
3. 选择主分支（main/master）作为部署源
